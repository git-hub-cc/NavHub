# NavHub 脚本说明文档

本目录包含用于维护和优化 NavHub 导航数据的一系列自动化脚本。

| 脚本名称                      | 作用描述                                                                                                                           |
| :---------------------------- | :--------------------------------------------------------------------------------------------------------------------------------- |
| `01process_icons.py`          | **指定文件图标处理**：针对特定 JSON 文件，批量下载外部图标 URL 并转换为本地 32x32 的 WebP 文件，更新路径引用。                     |
